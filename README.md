# Kanban Board

A modern React 18 + TypeScript Kanban board with accessible drag and drop, local persistence, JSON/CSV import and export, search, filters, light/dark mode, and realistic sample data.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Highlights

- Four workflow columns: To Do, In Progress, Review, Done
- Drag cards within and between columns with pointer, touch, and keyboard controls
- LocalStorage persistence with sample data fallback
- JSON and CSV import/export
- Search, priority filter, label filter, and clear/reset actions
- Responsive layout with light and dark mode
